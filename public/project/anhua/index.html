广州「AH创趣场」全案视觉与物料设计统筹

　　

　　　

　

角色： IP设计 / 平面设计 / LOGO设计

　

01. 空间视觉统筹

  • 视觉构建： 基于甲方对于此项目“吸引年轻人、主流动漫文化融合商场活力定位”的预期，结合我对动漫文化的理解，确立了高饱和但不刺眼的日系明亮色彩体系。
  • 物料设计与落地： 对场内从超大型空中吊幔、主题打卡墙，到精细的指引标识、活动宣传海报等超过 50 处物料进行了版面设计与适配。

　　

　

02. 品牌标识与字体设计

  • 融合商场方已有的视觉标志，加入二次元标志性的锐利切角与图形化元素，打造商场街道专属LOGO。
  • 在设计初期便充分考虑了线下物理空间的落地限制。最终输出的标准Logo完稿，完美适配了超过 10 种不同的线下制作材质工艺。

　

　　　

　　

03. 角色企划与技术落地

  • 前端模型引导： 基于甲方在项目中采用AIGC技术的需求，撰写精确的结构化Prompt，精准控制风格走向，确保产出符合特定受众人群喜好的日系赛璐璐风格。
  • 后端深度重制（核心环节）： 意识到AI生成的图像在商业落地存在诸多逻辑硬伤与版权局限性，为解决此问题与甲方需求期望之间的矛盾，我投入了大量精力（占比超总制作周期的 60%）进行人工重绘与修图。

　

04. 印前品控与周边材质实验

  • 印刷落地： 带领团队处理繁杂设计稿件的印前规范，严格确保文件提交印厂后的规范性。最终凭借严谨的印前检视理，实现了全场庞大物料的 0 印刷事故率与 0 返工率。
